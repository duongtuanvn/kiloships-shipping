# Kiloships Shipping Plugin

Plugin WordPress tích hợp Kiloships shipping label generation vào WooCommerce.

## Mô tả

Kiloships Shipping là plugin WordPress cho phép bạn tích hợp dịch vụ tạo nhãn vận chuyển Kiloships trực tiếp vào WooCommerce, giúp quản lý và tạo nhãn vận chuyển một cách dễ dàng.

## Tính năng

- Tích hợp API Kiloships
- Tạo nhãn vận chuyển từ đơn hàng WooCommerce
- Quản lý nhà cung cấp
- Báo cáo và thống kê
- Giao diện quản trị thân thiện

## Yêu cầu

- WordPress 5.0 trở lên
- WooCommerce 3.0 trở lên
- PHP 7.4 trở lên

## Cài đặt

1. Tải plugin về máy
2. Upload thư mục `kiloships-shipping` vào thư mục `/wp-content/plugins/`
3. Kích hoạt plugin qua menu 'Plugins' trong WordPress
4. Cấu hình plugin trong WooCommerce > Settings > Shipping

## Cấu hình

Sau khi kích hoạt plugin, bạn cần:

1. Vào WooCommerce > Settings > Shipping
2. Tìm phần Kiloships Shipping
3. Nhập API key của bạn
4. Cấu hình các tùy chọn vận chuyển
